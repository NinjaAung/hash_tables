# Time & Space Complexity For Each Of The Following

## Binary Tree

| ALgo   | Best Case | Worst Case |
|--------|-----------|------------|
| Space  | O(n)      | O(n)       |
| Access | O(log n ) | O(n)       |
| Search | O(log n)  | O(n)       |
| Insert | O(log n)  | O(n)       |
| Delete | O(log n)  | O(n)       |

## Hash-Table

| ALgo   | Best Case | Worst Case |
|--------|-----------|------------|

| Search | O(1)      | O(n)       |

| Delete | O(1)      | O(n)       |
